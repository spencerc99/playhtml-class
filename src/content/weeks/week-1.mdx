## Examples

### Example 1: a draggable object

This is the fastest way to make something feel shared and alive:

```tsx
export default function Room() {
  return (
    <main>
      <img
        id="stool"
        src="/red-stool.png"
        can-move=""
        draggable={false}
        style={{ width: 96, position: 'absolute', top: 120, left: 80 }}
      />
    </main>
  );
}
```

### Example 2: wrap the app with `PlayProvider`

`PlayProvider` turns on the shared PlayHTML features for the app:

```tsx
import { PlayProvider } from '@playhtml/react';

export function AppShell({ children }: { children: React.ReactNode }) {
  return (
    <PlayProvider
      initOptions={{
        cursors: { enabled: true },
      }}
    >
      {children}
    </PlayProvider>
  );
}
```
